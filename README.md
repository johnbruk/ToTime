# TOTIME Web App / PWA v0.9.2

Release dedicata al restyling grafico e allineamento UI al modello approvato.

## Novità principali
- Restyling grafico stile premium finance / Copilot Money-like.
- Tipografia più equilibrata e coerente.
- Home ridisegnata con card KPI, grafico andamento fatturato mese e menu card.
- Bottom navigation con icone e label più leggibili.
- Componenti card, form, badge e pulsanti più uniformi.
- Mantiene le funzionalità v0.9.1: Supabase, consuntivo manuale, spese di trasferta, template Fiscozen, tema chiaro/scuro ed export Excel.

## Deploy
Caricare su GitHub i file estratti, non lo ZIP. Netlify pubblicherà il nuovo commit.

## Cache
Il service worker è aggiornato a `totime-v092`. Se resta visibile una versione precedente, eseguire un refresh forzato o cancellare la PWA dalla schermata Home e aggiungerla nuovamente.
